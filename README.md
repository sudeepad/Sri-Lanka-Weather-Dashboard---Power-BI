# 🌦️ Sri Lanka Weather Dashboard (Power BI)

## 📊 Overview

This project is an interactive weather dashboard built using **Power BI** to visualize real-time and historical weather data across districts in Sri Lanka.

The dashboard provides insights into temperature, humidity, rainfall, and weather conditions, helping users analyze regional climate patterns effectively.

---

## 🎯 Features

* 🌍 District-wise weather visualization
* 🌡️ Temperature trends (Current & Historical)
* 🌧️ Rainfall & precipitation analysis
* 💨 Wind speed and humidity tracking
* 📍 Interactive filters by district
* 📅 Time-based analysis (daily / monthly trends)

---

## 🛠️ Tools & Technologies

* **Power BI Desktop**
* **DAX (Data Analysis Expressions)**
* **Power Query (ETL)**
* Weather API
